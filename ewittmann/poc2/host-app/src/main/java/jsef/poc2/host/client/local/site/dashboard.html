Extensible Application (POC2)

I am: GWT Host Application for POC #2

This is a host application written in GWT that has been designed to be extensible. The idea is that the S-RAMP Browser will be extensible in multiple ways. One extension point is the Artifact Details page's Tabs. By default, the Artifact Details page will have some built-in tabs such as Overview, Relationships, History, etc. Application Extensions should be able to add their own Tabs for specific artifact types. For example, a SwitchyardDocument should have a separate "Switchyard" tab, which displays Switchyard-specific information. This tab will be contributed by a runtime extension rather than built-in to the UI application. Further, the host application (S-RAMP Browser) will know nothing about the specifics of the tab or its content.
